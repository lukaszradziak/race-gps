## Requirements

* ESP32
* GPS 10Hz: VK2828U7G5LF

## Build board

* Install PlatformIO
* Platformio -> esp32dev -> Upload

## Frontend development

* Install node.js
* `cd frontend`
* `npm install`
* `npm run dev`
* Go to http://localhost:5173/
